# 행정서비스 통합제공시스템 구축(2단계 3차) - 정부24

## 개요

- **기간:** 2020.06 ~ 2021.03
- **유형:** 회사 프로젝트
- **인원:** 4명
- **역할:** 팀원
- **링크:** [정부24 온종일돌봄](https://www.gov.kr/portal/onestopSvc/alldaycare)

## 프로젝트 소개

> 정부24 행정서비스 통합제공시스템 구축 사업(2단계 3차) — 온종일돌봄 서비스 및 통계 기능 개발

행정서비스 포털(정부24)의 온종일돌봄 서비스 신청 기능과 원스톱 서비스 이용자 통계 기능을 개발한 프로젝트입니다.

## 기술 스택

| 분류 | 기술 |
|-----|-----|
| Backend | Java, Spring Framework, Mybatis |
| Frontend | JSP, jQuery |
| Database | Oracle |
| Infra | Tomcat, SVN |
| Tool | Eclipse |

## 주요 기능

- 온종일돌봄 서비스 신청 기능
- 행정전자서명(GPKI) 인증 적용
- 원스톱 서비스 이용자 통계

## 내가 기여한 부분

- 행정서비스 사용자 행정전자서명(GPKI) 인증 적용
- 온종일돌봄 서비스 개발
- 원스톱 서비스 이용자 통계 개발을 위한 SQL 운용

## 성과 / 배운 점

- 온종일돌봄 서비스 신청 및 서비스 이용자 통계 제공
