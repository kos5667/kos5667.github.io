# Batch를 이용한 공간/속성 GIS 데이터 ETL 구축 및 변동자료 처리

## 개요

- **기간:** 2021.03 ~ 2021.05
- **유형:** 회사 프로젝트
- **인원:** 7명
- **역할:** 개발팀 파트 담당

## 프로젝트 소개

> 내부 데이터 활용 및 고도화를 위한 GIS 데이터 ETL 구축 및 변동자료 처리 프로젝트

웹 크롤링과 Spring Batch를 활용하여 공간·속성 GIS 데이터를 추출·변환·적재하고, 변동 자료를 자동으로 처리하는 ETL 파이프라인을 구축하였습니다.

## 기술 스택

| 분류 | 기술 |
|-----|-----|
| Backend | Spring Boot, Java, Spring Batch, Scheduler, Jsoup, Apache POI |
| Database | PostgreSQL |
| Infra | CentOS, Tomcat, Jenkins |
| Build | Gradle |

## 주요 기능

- 변동 자료 처리
  - Jsoup을 사용해 웹에서 크롤링한 데이터를 정제하여 DW에 구축 및 적재
  - Spring Batch와 스케줄러를 활용하여 변동 자료를 처리하고 적절한 데이터 저장소에 적재
- 데이터 추출(공간, 속성)
  - 속성 데이터를 CSV 및 텍스트 파일 형식으로 추출 및 제공
  - 공간 데이터를 ESRI의 SHP 파일 형식으로 추출 및 제공

## 내가 기여한 부분

- 변동 자료 처리 파트 담당
- Jsoup 기반 웹 크롤링 및 데이터 정제 후 DW 적재
- Spring Batch와 스케줄러를 활용한 변동 자료 자동 처리
- 속성 데이터(CSV, 텍스트) 및 공간 데이터(SHP) 추출 기능 구현

## 성과 / 배운 점

- 정제된 데이터를 활용하여 서비스 다양성 확보
- 소규모 프로젝트 리더로서의 경험을 통해 책임감과 다양한 기술 지식 습득
