# 기술 스택

## Backend

- Java 21, Spring Boot 3.x, Spring MVC, Spring Security
- JPA, QueryDSL
- Node.js, TypeScript, Express
- Socket.io (WebSocket)

## Database & Messaging

- MySQL, PostgreSQL (PostGIS), Oracle, Tibero
- RabbitMQ (AMQP)
- Redis (캐시, 세션, 분산락)
- 파티션, 인덱스 설계, 쿼리 튜닝

## DevOps & Infra

- Kubernetes, Ingress, HPA
- CI/CD (Jenkins), 배포 자동화
- 배포 전략 (롤링, 카나리, 블루그린)
- Multi IDC, Multi Region 운영 경험

## Quality & Security

- Jest (TDD, 통합 테스트)
- 부하 테스트 수행 경험
- JWT, ACL, Vault, 암호화 적용
- 보안 점검 및 ISMS-P 대응 경험
